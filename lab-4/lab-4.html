Тема та постановка задачі

Тема: Адаптивна верстка за допомогою медіа-запитів.

Постановка задачі: Виконати адаптивну верстку для десктопної, планшетної та мобільної версії за допомогою медіа-запитів та CSS-правил.

Адаптивна верстка

Десктопна версія

Опис створення десктопної версії:

/* Десктопна версія */
@media (min-width: 1200px) {
    .container {
        max-width: 1200px;
    }
    .task-table th, .stats-table th {
        background-color: #8175d3;
    }
}

Планшетна версія

Опис створення планшетної версії:

/* Планшетна версія */
@media (min-width: 768px) and (max-width: 1199px) {
    .container {
        max-width: 768px;
    }
    .task-table th, .stats-table th {
        background-color: #ccffcc;
    }
}

Мобільна версія

Опис створення мобільної версії:

/* Мобільна версія */
@media (max-width: 767px) {
    .container {
        max-width: 100%;
        padding: 10px;
    }
    .task-table th, .stats-table th {
        background-color: #07077a;
    }
}

Логічні оператори

Використання логічних операторів у процесі адаптивної верстки:

/* Приклад використання логічних операторів */
.container {
    width: 100%;
    max-width: 1200px;
    background: #fff;
    padding: 20px;
    border-radius: 8px;
    box-shadow: 0 4px 8px rgba(0,0,0,0.1);
}

@media (min-width: 1200px) {
    .container {
        max-width: 1200px;
    }
}

@media (min-width: 768px) and (max-width: 1199px) {
    .container {
        max-width: 768px;
    }
}

@media (max-width: 767px) {
    .container {
        max-width: 100%;
        padding: 10px;
    }
}
